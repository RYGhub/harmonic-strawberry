Harmonic Strawberry

Discord bot for Impressive Strawberry

About

Harmonic Strawberry is a Discord bot which allows moderators of a server to create server-wide achievements which can then be awarded to the server members.

➕ Add to your server

Work in progress

Currently, the bot is not publicly available; you'll need to host your own instance of both Impressive Strawberry and Harmonic Strawberry.

Contact the developers

To contact the developers of Harmonic Strawberry, please visit the GitHub repository:

  • if you encountered a bug, please open a new issue;
  • if you want to talk about the project or just want to ask a question, create a new discussion;
  • if you want to contribute to the project, please open a new pull request!

Commands list

/create {crystal} {...} Create a new achievement
/delete {crystal} Delete an achievement
/edit {crystal} [...] Edit an achievement
/list List all achievements of the server
/setup {channel} {role} Setup notifications in a channel and allow role to manage achievements
/unlock {user} {crystal} Unlock an achievement for a member
/view [user] View the achievements unlocked by a member
